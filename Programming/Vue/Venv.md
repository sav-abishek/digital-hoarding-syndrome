## Virtual environment from CLI [Reference](https://nordicapis.com/how-to-create-an-api-using-the-flask-framework/)

##### Install Virtual env:
	pip install virtualenv

#### Create a new folder and enter in
	mkdir <Folder_name>
	cd <folder_name>

##### Convert the folder into a virual env:
	virtualenv .

##### Activate the virtual env:
	Scripts\activate

##### Deactivate the virtual env:
	deactivate
